# Trial of the Outer Spiral

## Requirements
  * Python 3
  * numpy
  * pygame
  * python-tcod
  * Microsoft Visual C++ Redistributable or GCC (for tcod)
 
## Compatibility
I have tested running the game on Fedora 29 and Winodws 10. I expect it to run on any operating system
